# Domínio do projeto

O glossário do domínio (termos como Prévia Operacional, Central de Resultados,
Contrato Semântico, Catálogo de Dados, Linhagem, Pré-filtro de Páginas,
Idempotência, Indicador) está em [`../CONTEXT.md`](../CONTEXT.md).

Use os termos do `CONTEXT.md` em código, commits, PRD e backlog. Se um termo
novo for necessário, atualize o `CONTEXT.md` antes de usá-lo em outros
documentos.
